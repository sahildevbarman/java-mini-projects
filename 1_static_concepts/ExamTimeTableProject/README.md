# College Exam Time Table Project

This is a small Java real-life mini project.

## Concepts Used
- static class-level variables
- arrays
- classes and objects
- real-life modeling

## Description
Exam time table is common for all students of B.Tech. II Year,
so it is implemented using a static array at class level.

## How to Run
1. Compile all `.java` files
2. Run `Main.java`
